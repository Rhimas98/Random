# Coefficients

Tranfer function in discrete time
TF = d0*z / (z + c0)

## Air model

### Without filtering
	- c0 = 0.8202
	- d0 = 0.3909
	- pole = 0.8208 (3.15 Hz)

### With filtering
	- c0 = -0.7704
	- d0 = 0.4976
	- pole = 0.7704 (4.15 Hz)
	- cutoff filter = 4.775 Hz 

## Ground model

	- d0 = 0.3909
	- pole = 0.8532 (2.53 Hz)

### With filtering
	- c0 = -0.8411
	- d0 = 0.3276
